## The Financial Layer for the Agentic Economy

SOHO is an on-chain credit-backed payment protocol designed specifically for autonomous AI agents. We are moving beyond human-centric checkout flows to create a world where agents can discover, negotiate, and settle transactions independently.

## The Product

### **x402: The Agent Payment Standard**
Current HTTP status codes weren't built for machines. SOHO champions the **x402 standard**—an extension of HTTP 402 "Payment Required"—that enables a seamless, stateless handshake between an AI agent and a merchant. No carts, no "Check Out" buttons, just machine-to-machine settlement.

### **SoHo Pay: Programmable Credit**
Agents shouldn't need a pre-funded wallet for every task. **SoHo Pay** provides a non-custodial credit layer that allows agents to borrow liquidity on-demand from a SOHO Vault to settle transactions instantly.

### **Clawbot: The Reference Agent**
Clawbot is our flagship agent implementation. It demonstrates the full power of the protocol by autonomously identifying payment requirements and executing settlement via the SoHo credit layer.

## Key Features

* **Agent-Native Payments**: Purpose-built for high-frequency, low-cost autonomous transactions.
* **Non-Custodial Architecture**: Powered by **EIP-712** delegated signing. Users maintain full control of their private keys at all times.
* **On-Chain Guardrails**: Real-time spending limits and session-based approvals are enforced by the protocol to ensure agent safety.
* **Instant L2 Settlement**: Fully operational on **Base Sepolia** for sub-second finality and minimal gas fees.

## The Ecosystem

SOHO connects four critical participants in the digital economy:

1.  **AI Agents**: Use SOHO Skills to perform economic actions.
2.  **Merchants**: Integrate the **Merchant SDK** to accept x402 payments.
3.  **Users**: Delegate spending authority within safe, pre-defined boundaries.
4.  **Liquidity Providers**: Supply assets to SOHO Vaults to earn yield from agentic commerce.

## The AI Commerce Flywheel
As more agents adopt the x402 standard, merchant adoption grows, creating more transaction volume and deeper liquidity. SOHO sits at the center of this self-reinforcing growth cycle, providing the foundational infrastructure for the next phase of the internet.
